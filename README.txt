Thank you for installing DeskSearch!

To use:

1. Open desksearch and it will appear in the bottom right of your monitor
2. Enter a url into the text box and press "GO"
3. "https" will be added automatically if it is not already in the URL

If you recieve any errors while using DeskSearch then please report it to the Discord: https://discord.gg/vgGVHhfBcf
